# Simple TODO on Golang

### Функции
- Получать аргументы из терминала в формате флагов (см. раздел "Флаги для терминала")
- Без указания аргументов выводить красивое меню для работы с TODO
- Возможность выводить все задачи, добавлять, редактировать и удалять задачу, выходить из программы
- Сама задача может быть отмечена, как выполненная, но по умолчанию статус "Не выполнена"

### Декомпозиция цели
0. Создание структуры самой задачи, определение архитектуры, набора функций
1. Создание меню-интерфейса
2. Создание функций работы с задачамами (CRUD)
3. Реализация отметки задач
4. Создание CLI-взаимодействия
