# Simple TODO on Golang

### Функции
- Выводить красивое меню для работы с TODO
- Возможность выводить все задачи, добавлять, редактировать и удалять задачу, выходить из программы
- Сама задача может быть отмечена, как выполненная, но по умолчанию статус "Не выполнена"
- Правильное индексирование и проверка всех ошибок в течение выполнения программы
